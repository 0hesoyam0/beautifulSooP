SOOP Language Documentation

Key Words

  • namespace - Define a namespace for classes and interfaces.
  • class - Declare a class.
  • interface - Declare an interface.
  • def - Declare a method.
  • if, else - Conditional statements.
  • for, in, while - Looping constructs.
  • switch, case - Multi-way selection.
  • return - Return from a method.
  • public, private, protected - Access modifiers.

Language Constructs

Namespaces

namespace MyProject {
    class MyClass {
        // ...
    }
}

Classes

class MyClass {
    public def myMethod(arg1: int, arg2: string) : int {
        // Method implementation
    }
}

Interfaces

interface IMyInterface {
    def myMethod(arg1: int) : int;
}

Methods

class MyClass {
    public def myMethod(arg1: int, arg2: string) : int {
        // Method implementation
        return 42;
    }
}

Control Structures

if (x > 0) {
    // Execute if x > 0
} else {
    // Execute if x <= 0
}

for (i in 0..9) {
    // Iterate from 0 to 9
}

while (x < 100) {
    // Execute while x < 100
}

switch (x) {
    case 1:
        // Execute if x == 1
    case 2:
        // Execute if x == 2
    default:
        // Execute if none of the above conditions match
}

Access Modifiers

class MyClass {
    public x: int = 0; // Accessible from anywhere
    private y: int = 0; // Accessible only within the class
    protected z: int = 0; // Accessible within the class and derived classes
}
